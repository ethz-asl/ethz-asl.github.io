Experiment data supporting the IEEE-RAM 2022 paper on autonomous balloon navigation. Includes indoor and outdoor flight trials, flight logs, and CAD files to replicate the balloon systems. Preview imagery to be rehosted with the dataset.

![Outdoor balloon prototype](/assets/datasets/ram_22_autonomous_ballooning/outdoor_prototype.png)

## Dataset Description

- Two zip archives covering indoor and outdoor experiments
- Contains flight videos, logged flight data, and CAD designs for the balloon platform
- Reference implementation: [project_balloon](https://github.com/simonjeger/project_balloon)

## Downloads

- Indoor experimental data (includes CAD): <https://doi.org/10.3929/ethz-c-000787616>
- Outdoor experimental data (includes CAD): <https://doi.org/10.3929/ethz-c-000787616>
